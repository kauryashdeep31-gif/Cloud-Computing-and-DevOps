# Day 6 – Cloud Computing Fundamentals (07/07/2026)

## Overview

Today's learning session focused on understanding the fundamentals of Cloud Computing and its importance in modern software development. Cloud computing has transformed the way organizations store data, deploy applications, and manage computing resources by providing services over the internet. I explored the basic architecture of cloud computing, its key characteristics, and the advantages it offers over traditional on-premise infrastructure.

## Topics Covered

### 1. Introduction to Cloud Computing

* Definition of Cloud Computing
* Evolution of cloud technology
* Basic cloud architecture
* Importance of cloud computing in modern IT

### 2. Characteristics of Cloud Computing

* On-demand self-service
* Broad network access
* Resource pooling
* Rapid elasticity
* Measured service

### 3. Benefits of Cloud Computing

* Cost efficiency
* Scalability
* High availability
* Flexibility
* Reliability
* Automatic updates
* Improved collaboration
* Disaster recovery support

### 4. Introduction to AWS

I learned about Amazon Web Services (AWS), one of the most widely used cloud platforms. The session covered:

* What AWS is
* Advantages of AWS
* Popular AWS cloud services
* Why organizations prefer AWS for cloud solutions

### 5. Services and Web Services

I studied the concept of services and how web services enable communication between different applications over the internet. I also learned how web services improve interoperability between systems.

### 6. Service-Oriented Architecture (SOA)

The session introduced Service-Oriented Architecture (SOA), where software applications are built using reusable and independent services. I understood how SOA improves scalability, maintainability, and communication among distributed applications.

### 7. Cloud Platforms

I explored different cloud platforms used by organizations to build and deploy cloud-based applications. Each platform provides tools and services that simplify application development and infrastructure management.

### 8. Cloud Service Models

#### Infrastructure as a Service (IaaS)

* Provides virtual machines, storage, and networking
* Users manage the operating system and applications
* Offers flexible and scalable infrastructure

#### Platform as a Service (PaaS)

* Provides a complete development platform
* Developers can build, test, and deploy applications without managing servers

#### Software as a Service (SaaS)

* Software is delivered through the internet
* No installation or maintenance is required on the user's device

#### Function as a Service (FaaS)

* Executes individual functions in a serverless environment
* Infrastructure management is handled automatically by the cloud provider

## Learning Outcome

Today's session strengthened my understanding of cloud technologies, cloud architecture, AWS fundamentals, SOA, web services, and the major cloud service models. I now have a clear understanding of how cloud computing simplifies application deployment, improves scalability, reduces infrastructure costs, and supports modern software development.
