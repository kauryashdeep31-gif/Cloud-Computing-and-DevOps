### Linux Commands

Practiced basic Linux commands for navigating and managing files and directories:

* `pwd` – Display the current working directory.
* `ls` – List files and directories.
* `cd` – Change the current directory.
* `mkdir` – Create a new directory.
* `cp` – Copy files and directories.
* `mv` – Move or rename files.
* `rm` – Remove files and directories.

* Learned the basics of Continuous Integration (CI) and Continuous Deployment (CD).

### Windows File Structure

* Compared the basic organization of Windows and Linux file systems.

### Types of Software

Learned the differences between:

* **System Software** – Manages hardware and system resources (e.g., Linux, Windows).
* **Utility Software** – Maintains and optimizes the system (e.g., Antivirus, Backup tools).
* **Application Software** – Helps users perform specific tasks (e.g., Chrome, MS Word).

### Linux File System

Studied the purpose of important Linux directories:

* `/home` – User files and personal directories.
* `/etc` – System configuration files.
* `/usr` – User applications and utilities.
* `/var` – Log files and variable data.
* `/bin` – Essential system commands.
