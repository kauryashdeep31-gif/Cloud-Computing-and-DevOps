Day 1: Introduction to Cloud Computing and DevOps
Overview

Today I started learning the fundamentals of Cloud Computing and DevOps. The goal was to understand how modern applications are hosted, managed, and scaled using cloud technologies.

Topics Covered
1. Cloud Computing

Cloud Computing is the delivery of computing services such as servers, storage, databases, networking, and software over the internet. Instead of owning physical hardware, users can access resources on demand.

2. Major Cloud Service Providers
AWS (Amazon Web Services) – Market leader with a wide range of cloud services.
Microsoft Azure – Popular among enterprises and organizations using Microsoft products.
Google Cloud Platform (GCP) – Known for data analytics, AI, and machine learning services.
3. Advantages of Cloud Computing
Auto Scaling

Auto Scaling automatically increases or decreases computing resources based on demand. This improves performance and reduces costs.

Pay-as-You-Go Model

Users only pay for the resources they consume, eliminating the need for large upfront infrastructure investments.

4. Comparison of Computing Models
Type	Description
Standalone Computing	A single computer works independently without network dependency.
Client-Server Computing	Clients request services from a centralized server.
Distributed Computing	Multiple computers work together to perform tasks.
Cloud Computing	Resources are delivered over the internet on demand.
5. Basic Cloud Architecture
User → Internet → Cloud → Application → Database
Working:
User sends a request.
Request travels through the internet.
Cloud infrastructure receives and processes it.
Application handles business logic.
Database stores and retrieves required data.
Response is returned to the user.
